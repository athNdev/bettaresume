# Exporting to PDF

Create professional, print-ready PDF documents from your resume.

## Quick Export

1. Click the **Export** button in the toolbar
2. Choose **PDF** format
3. Your file downloads automatically

## What's Included

- All visible sections in order
- Applied template styling
- Custom colors and fonts
- Proper page formatting

## Best Practices

### Before Exporting

- **Preview First** — Check the live preview panel
- **Verify All Sections** — Ensure content is complete
- **Check Page Length** — Aim for 1-2 pages

### File Naming

Save with descriptive names:
- `john-doe-resume-2024.pdf`
- `john-doe-software-engineer.pdf`
- `john-doe-google-application.pdf`

## Different Versions

Export different versions for different purposes:

| Version | Use Case |
|---------|----------|
| Main resume | General applications |
| Company-specific | Tailored applications |
| Skills-focused | Technical roles |

## Troubleshooting

### PDF Looks Different Than Preview

- Ensure browser zoom is at 100%
- Try a different browser
- Check for long text that may wrap

### File Too Large

- Reduce content if extensive
- Template choice affects file size
- Remove optional sections if needed
