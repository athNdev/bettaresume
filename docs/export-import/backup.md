# Backup Your Resume

Export your resume data as JSON to create backups and share between devices.

## Creating a Backup

1. Click the **Export** button in the toolbar
2. Choose **JSON** format
3. Save the `.json` file somewhere safe

## What's Included in the Backup

- All resume content
- Section order and visibility
- Custom styling settings
- All versions and their content
- Template selection

## Backup Recommendations

### When to Backup

- Before major changes
- After completing a version
- Weekly during active job searching
- Before switching devices

### Where to Store Backups

| Location | Pros |
|----------|------|
| Cloud storage (Google Drive, Dropbox) | Access anywhere |
| Email to yourself | Searchable, timestamped |
| Local folder | Quick access |

### Naming Your Backups

Use descriptive, dated names:
```
resume-backup-2024-01-15.json
resume-backup-pre-major-edit.json
resume-backup-complete.json
```

## Restoring from Backup

1. Go to the dashboard
2. Click the **Import** button
3. Select your JSON backup file
4. Resume is restored with all data

## Multiple Resumes

Each JSON file is a complete resume. You can:
- Maintain separate backups for different resumes
- Share backups between computers
- Keep historical versions for reference
