# Importing Resume Data

Restore backups or migrate data from JSON files.

## How to Import

### From the Dashboard

1. Go to the **Dashboard**
2. Click the **Import** button (or look for upload icon)
3. Select a JSON file
4. Resume appears in your list

### Supported Format

Import files must be JSON format exported from Betta Resume.

## What Gets Imported

| Data | Imported |
|------|----------|
| All content | ✅ |
| Section order | ✅ |
| Styling settings | ✅ |
| All versions | ✅ |
| Template choice | ✅ |

## After Importing

- Verify all sections loaded correctly
- Check that styling looks right
- Review any versions that were included
- Make a backup of your current data first (recommended)

## Importing Tips

### Before You Import

- **Backup first** — Export your current data
- **Check the file** — Ensure it's a valid JSON
- **One at a time** — Import resumes individually

### Common Issues

**Import fails silently**
- File may be corrupted or wrong format
- Try opening the JSON file in a text editor to verify

**Missing content after import**
- Check if the export was complete
- Try re-exporting from the source

**Duplicate resume**
- Imported resumes are added as new entries
- Delete duplicates from the dashboard if needed

## Transferring Between Devices

1. Export as JSON on device A
2. Transfer the file (email, cloud, USB)
3. Import on device B

Your resume is now available on both devices.
