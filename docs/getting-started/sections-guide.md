# Resume Sections Explained

Your resume is built from various sections, each serving a specific purpose.

## Personal Info 👤

Your contact details and professional identity:
- Full name and professional title
- Email, phone, location
- LinkedIn, GitHub, website links

## Summary 📝

A brief professional overview:
- 2-4 sentences about your expertise
- Supports rich text formatting
- First thing recruiters read — make it count!

## Experience 💼

Your work history:
- Job title, company, dates
- Location and employment type
- Bullet points for achievements
- Technologies used

## Education 🎓

Academic background:
- Degree, institution, dates
- GPA, honors, relevant coursework
- Certifications can go here or in a separate section

## Skills ⚡

Technical and soft skills:
- Organize by category (Languages, Frameworks, Tools, etc.)
- Optional proficiency levels
- Keep relevant to target job

## Projects 🚀

Showcase your work:
- Project name and description
- Technologies used
- Links to live demo or GitHub

## Additional Sections

- **Certifications** 📜 — Professional credentials with expiry dates
- **Awards** 🏆 — Recognitions and achievements
- **Languages** 🌍 — Language proficiencies
- **Volunteer** 🤝 — Community involvement
- **Publications** 📚 — Papers and articles
- **References** 📋 — Available upon request
