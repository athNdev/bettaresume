# Saving Your Changes

Betta Resume uses **explicit saving** for important changes.

## How It Works

1. Make your edits in the form
2. A save bar appears at the bottom when you have unsaved changes
3. Click **"Save Changes"** to commit
4. Or **"Discard"** to revert to the last saved state

## Why Explicit Saving?

- **Prevents accidental changes** — no surprise edits
- **Creates clear activity history** — see what changed when
- **Lets you review before committing** — catch mistakes
- **Easy to undo** — just discard before saving

## Activity Log

Every save is logged in the Activity section:

- What section was changed
- What fields were modified
- When the change was made

This helps you track your editing history.

## What Auto-Saves

Some changes save automatically (no save button needed):

- Section reordering (drag and drop)
- Section visibility toggles
- Template switching
- Settings changes

## Best Practices

- Save after completing each section
- Review the preview before saving
- Use "Discard" if you make mistakes
- Check the Activity log to see your history
