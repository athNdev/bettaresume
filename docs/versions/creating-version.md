# Creating a Version

There are two ways to create a tailored version of your resume.

## Method 1: From the Versions Panel

1. Open your main resume in the editor
2. In the left panel, find the **"Versions"** section
3. Click **"+ New Version"**
4. Choose a domain (Software, Data Science, Product, etc.)
5. Give it a descriptive name

## Method 2: From a Section (Quick Method)

1. While editing any section, click the **"Create Tailored Copy"** button
2. This creates a version with that section already customized
3. Perfect when you know exactly what to change

## Naming Best Practices

Good version names help you stay organized:

| ✅ Good | ❌ Avoid |
|---------|----------|
| "Frontend Developer - Startup" | "Version 2" |
| "Google SWE Application" | "Copy" |
| "Data Science - ML Focus" | "New Resume" |

## What Happens When You Create a Version

- All sections start **linked** to the main resume
- The version inherits all content from main
- You can then customize specific sections as needed
- Linked sections will auto-update when main changes
