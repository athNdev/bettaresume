# Dashboard Overview

Your command center for managing all your resumes.

## Main Features

### Resume Cards

Each resume shows:
- **Name** — Your name or resume title
- **Last edited** — When you last worked on it
- **Version count** — How many versions exist
- **Template** — Visual preview of the design

### Quick Actions

On each resume card:
- **Edit** — Open in the editor
- **Duplicate** — Create a copy
- **Archive** — Hide from main view
- **Delete** — Remove permanently

## View Options

### Grid View
- Visual card layout
- See template previews
- Best for browsing

### List View
- Compact rows
- More info visible
- Best for many resumes

Toggle between views using the view switcher in the toolbar.

## Dashboard Stats

At a glance information:
- Total resumes
- Recently edited
- Version count across all resumes

## Creating New Resumes

Click the **+ New Resume** button to start fresh.

You'll be taken to the editor with a blank template ready to fill in.

## Organization Tips

### Use Clear Names
Name resumes descriptively:
- "Software Engineer - General"
- "Product Manager - Startup Focus"
- "2024 Job Search Master"

### Archive Old Resumes
Keep your dashboard clean by archiving resumes you're not actively using. They're still accessible but hidden from the main view.
