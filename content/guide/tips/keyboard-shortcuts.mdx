# Keyboard Shortcuts

Speed up your workflow with these shortcuts.

## Editor Shortcuts

| Action | Windows | Mac |
|--------|---------|-----|
| Save | Ctrl+S | ⌘+S |
| Undo | Ctrl+Z | ⌘+Z |
| Redo | Ctrl+Shift+Z | ⌘+Shift+Z |
| Bold text | Ctrl+B | ⌘+B |
| Italic text | Ctrl+I | ⌘+I |

## Navigation

| Action | Shortcut |
|--------|----------|
| Toggle preview panel | Click the preview toggle |
| Switch sections | Click in section list |
| Expand/collapse sections | Click the arrow |

## Rich Text Editor

When editing text content:

| Action | Windows | Mac |
|--------|---------|-----|
| Bold | Ctrl+B | ⌘+B |
| Italic | Ctrl+I | ⌘+I |
| Underline | Ctrl+U | ⌘+U |
| Link | Ctrl+K | ⌘+K |
| Bullet list | Start line with "- " | Start line with "- " |

## Tips

### Quick Navigation

- Use the section sidebar to jump between sections
- Collapse sections you're not editing
- Keep the preview panel visible for real-time feedback

### Efficient Editing

- Tab between form fields
- Use keyboard shortcuts for text formatting
- Drag and drop for reordering

## Browser Shortcuts

These work in most browsers:

| Action | Windows | Mac |
|--------|---------|-----|
| Refresh page | F5 or Ctrl+R | ⌘+R |
| Zoom in | Ctrl++ | ⌘++ |
| Zoom out | Ctrl+- | ⌘+- |
| Reset zoom | Ctrl+0 | ⌘+0 |
